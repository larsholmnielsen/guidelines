13. Size (O)
^^^^^^^^^^^^

Unstructured size information about the resource (occurrences: 0-n).

**Allowed values, examples, other constraints**

Free text.

Examples: "15 pages", "6 MB"

Example
~~~~~~~
.. code-block:: xml
   :linenos:

   <sizes>
     <size>15 pages</size>
     <size>6 MB</size>
   </sizes>
